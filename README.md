# webx preloader v1.0.0

Minimalist entry screen with multi-layer bot protection and anti-tamper system.

### Features
- **Bot Detection:** System checks for webdriver, headless mode, and hardware spoofing.
- **Anti-Tamper:** Built-in console branding and debugger traps for F12 inspector.
- **Dynamic Progress:** Randomized loading segments with percentage tracking.
- **Haptic Vibe:** Mobile vibration feedback on successful entry.
- **Smooth Exit:** Cinematic blur and scale-down transition.

### Setup
1. Place your animation in the project folder as `cat.gif`.
2. Open `index.html` and set your URL in the `CONFIG` object:
   `URL: "https://your-website.com"`
3. Customize colors in the `:root` CSS block.

### Deployment
1. Push files to your repository.
2. Go to **Settings** -> **Pages**.
3. Set **Branch** to `main` and click **Save**.
