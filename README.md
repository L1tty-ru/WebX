# webx preloader v1.2.0

Minimalist entry screen with multi-layer bot protection and anti-tamper system.

### Features
- **Smart Detection:** System checks for webdrivers, headless mode, and hardware spoofing.
- **Shy Mode:** Loading auto-pauses on F12. The cat gets shy, progress stops, debugger loop.
- **Dynamic Progress:** Randomized loading segments with percentage tracking.
- **Haptic Vibe:** Mobile vibration feedback on successful entry.
- **Cinema Exit:** Smooth blur and scale-down transition.

### Setup
1. Drop your animations into the `assets` folder as `cat.gif` and `shy.gif`.
2. Open `index.html` and set your URL in the `cfg` object:
   `url: "https://your-website.com"`
3. Customize colors in the `:root` CSS block.

### Deployment
1. Push files to your repository.
2. Go to **Settings** -> **Pages**.
3. Set **Branch** to `main` and click **Save**.
