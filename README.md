# webx preloader 🐈‍⬛

Entry screen with built-in bot protection.

### Setup
1. Place your animation in the project folder as `cat.gif`.
2. Open `index.html` and edit the `CONFIG` object at the top of the script:
   `YOUR_SITE_URL: "https://your-website.com"`
3. Colors can be customized in the `:root` CSS block.

### Features
* **Bot Protection:** Detects automation and headless environments.
* **Smooth Motion:** Breathing effect and seamless status transitions.
* **Fast Entry:** 0.3s quick-in animation.

### Deployment
1. Push files to your repository.
2. Go to **Settings** -> **Pages**.
3. Set the Branch to `main` and click **Save**.
